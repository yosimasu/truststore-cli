# Project Brief: truststore cli

- truststore example.org
- truststore example.org:443
- truststore list example.org
- truststore list example.org:443
  - list certificate chain of example.org

- truststore truststore.pem
- truststore truststore.jks -p ...
- truststore truststore.p12 -p ...
- truststore list fullchain.pem
- truststore list truststore.jks -p ...
- truststore list truststore.p12 -p ...
  -  list certificate chain of the truststore file

- truststore add example.org --target fullchain.pem
- truststore add example.org --target fullchain.jks --target-password ...
- truststore add example.org:443 --target fullchain.p12 --target-password ...
- truststore add truststore.pem --target fullchain.p12 --target-password ...
- truststore add truststore.jks --password --target fullchain.p12 --target-password ...
- truststore add truststore.p12 --password --target fullchain.p12 --target-password ...
  - add cacert to the truststore file

- truststore rm example.org --target fullchain.pem
- truststore rm example.org --target fullchain.jks --target-password ...
- truststore rm example.org:443 --target fullchain.p12 --target-password ...
- truststore rm truststore.pem --target fullchain.p12 --target-password ...
- truststore rm truststore.jks --password --target fullchain.p12 --target-password ...
- truststore rm truststore.p12 --password --target fullchain.p12 --target-password ...
  - remove cacert to the truststore file

- binary command line can run on macOS, Linux, Windows
